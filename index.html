Projetos em Ciência de Dados

Modelo de Classificação - Plantas Iris

Projeto para praticar meus estudos em algoritmos de classificação. O projeto consiste em fazer uma análise comparativa entre os algoritmos de classificação com Regressão Linear e Rede Perceptron. Um projeto básico para demonstrar de forma didática uma introdução sobre Machine Learning.

As ferramentas utilizadas foram:

  • Python, Pandas, Matplotlib e Seaborn.
  • Jupyter Notebook.
  • Sklearn
  • Linear Regression
  • Perceptron

Modelo de Credit Scoring - Regressão Logística

Projeto do Curso de Credit Score da Alura. Criação de um Modelo de Credit Score utilizando Regressão Logística para classificar um tomador de crédito como adimplentes ou inadimplentes, ou seja, uma probabilidade de 0 ou 1. Tendo como base de dados da 'Germam Credit Data' encontrada no site da UCI machine learning repository. Foi aplicado conhecimentos estatísticos de matriz de confusão, métricas de acucária, precisão e recall, curva ROC.

As ferramentas utilizadas foram:

  • Python
  • Bibliotecas: Pandas, Matplotlib, Seaborn
  • Jupyter Notebook
  • Sklearn, Logistic Regression
