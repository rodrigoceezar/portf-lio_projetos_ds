Modelo de Regressão

Projeto do Curso de Credit Score da Alura. Criação de um Modelo de Credit Score utilizando Regressão Logística para classificar um tomador de crédito como adimplentes ou inadimplentes, ou seja, uma probabilidade de 0 ou 1. Tendo como base de dados da 'Germam Credit Data' encontrada no site da UCI machine learning repository. Foi aplicado conhecimentos estatísticos de matriz de confusão, métricas de acucária, precisão e recall, curva ROC.

As ferramentas utilizadas foram:

  • Python
  • Bibliotecas: Pandas, Matplotlib, Seaborn
  • Jupyter Notebook
  • Sklearn, Logistic Regression
  • SAIBA MAIS
